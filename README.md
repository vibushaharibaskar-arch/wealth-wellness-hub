# Wealth Wellness Hub

## 1. Project Overview
> Wealth Wellness Hub is a digital platform designed to help individuals manage both **financial health and mental wellbeing related to money**. It combines financial planning tools with behavioral insights to guide users through life events, financial stress, and long-term wealth building.

---

## 2. Problem Statement
- Many people struggle with **financial anxiety and uncertainty about future planning**.
- Existing apps focus only on **investment tracking or budgeting**, but not **financial wellbeing**.
- Users lack tools to prepare for **life shocks such as job loss, inflation, or market crashes**.

---

## 3. Our Solution
Wealth Wellness Hub provides:
- Personalized financial insights
- Life event simulations
- Behavioral nudges for better money habits
- A holistic view of financial and mental wellbeing

---

## 4. Key Features
**Life Shock Simulator**
- Simulates financial events like:
  - Job loss
  - Market crash
  - Medical emergency
  - Inflation spike

**Financial Health Score**
- AI-driven score combining:
  - Savings ratio
  - Debt level
  - Emergency funds
  - Risk tolerance

**Wellness Nudges**
- Personalized reminders to reduce financial stress.

**Goal Tracking**
- Tracks progress for goals like:
  - buying a house
  - retirement
  - travel fund

---

## 5. Target Users
- Young professionals
- Students entering the workforce
- Individuals with financial anxiety
- Users planning long-term financial goals

---

## 6. Tech Stack
- Frontend: React / HTML / CSS
- Backend: Python / Node.js
- Database: Firebase / MongoDB
- Visualization: Chart.js
- Deployment: Vercel / GitHub Pages
